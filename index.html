Como usar?

Passo 1: Preparação dos Dados

  1. Formato do Arquivo: O programa suporta arquivos no formato XLSX. Certifique-se de ter seus dados organizados em uma planilha neste formato.

Passo 2: Carregamento do Arquivo

  1. Seleção do Arquivo: Clique no botão "Selecionar Arquivo" para escolher o arquivo que contém seus dados.
  2. Análise dos Dados: Após selecionar o arquivo, o programa irá processar os dados automaticamente e apresentar as perguntas disponíveis em um seletor.

Passo 3: Visualização dos Gráficos

  1. Escolha uma Pergunta: Use o seletor para escolher a pergunta que deseja analisar.
  2. Visualização dos Resultados: O programa irá gerar automaticamente um gráfico adequado para a pergunta selecionada. Se a pergunta for a última da lista, uma nuvem de palavras será exibida.

Passo 4: Exploração Adicional

  1. Alternar Entre Perguntas: Você pode alternar entre as perguntas usando o seletor a qualquer momento para explorar diferentes aspectos dos dados.
  2. Download de Gráficos: Se desejar, é possível fazer o download dos gráficos gerados para uso posterior.

Passo 5: Interpretação dos Resultados

  1. Análise dos Gráficos: Utilize os gráficos gerados para compreender melhor a distribuição e padrões dos seus dados.
  2. Tomada de Decisões: Com base na análise dos gráficos, tome decisões ou extraia insights relevantes para o seu projeto ou negócio.
